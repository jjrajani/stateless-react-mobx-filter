#### Stateless React Components with Mobx and Mobx-React

### Installing
  `git clone https://github.com/jjrajani/stateless-react-mobx-filter.git`

  `cd classless`

  `yarn install`

  `yarn start`


### Folder Structure

After creation, your project should look like this:

```
classless/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    app/
      App.js
      App.test.js
      App.css
      components/
        filter/
          fitler.js
        user/
          User.js
          user.css
          components/
            FilterByAgeUserList.js
            UserList.js
            userList.css
          utils/
            createUser.js
            users.js
    index.js
    index.css
```
